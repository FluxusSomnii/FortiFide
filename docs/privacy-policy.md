# Forti Fide — Privacy Policy

*Effective date: upon first release.*

## The one-sentence version

Forti Fide processes all data locally on your device and does not collect, transmit, or store any personal data on external servers.

## What Forti Fide processes

| Data type | What happens |
|-----------|-------------|
| Audio from microphone or system | Processed locally by Whisper. Converted to text. Saved locally if Rec Audio is enabled. Never transmitted. |
| Transcript text | Stored locally. Never transmitted. |
| Pattern annotations | Generated locally. Stored locally. Never transmitted. |
| Check-in state data | Stored locally. Never transmitted. |
| Session metadata | Stored locally. Never transmitted. |
| AI analysis | If you use AI Analysis, transcript excerpts are sent to the Claude API (Anthropic). See below. |

## What Forti Fide does not do

- Does not collect analytics or usage data
- Does not require an account
- Does not transmit your audio, transcripts, or annotations to any external server
- Does not include advertising
- Does not sell, license, or share your data
- Does not have telemetry

## The Claude API

The AI Analysis feature sends transcript text to the Claude API (Anthropic). This only occurs when you explicitly press the Analyse button. It does not occur automatically. Anthropic's privacy policy governs what happens to data sent to the API. If you prefer, configure a local model in Settings — when a local model is configured, no data leaves your device for any reason.
