# Forti Fide — Visual Philosophy

*Lucid Depth.*

## The principle

Transparency is not a theme applied decoratively. It is the actual design principle. An interface that has nothing to hide looks like one that has nothing to hide.

## Visual language

- **Dark, not black** — Background #0c0c12. The colour of depth you can see through rather than into.
- **Glass morphism done right** — Surfaces at rgba(255,255,255,0.04) with 1px borders at rgba(255,255,255,0.10). Depth that reveals rather than hides.
- **Thin borders** — rgba(255,255,255,0.07). Barely visible. Structure without opacity.
- **Motion only when communicative** — Nothing bounces. Nothing performs. Motion is the motion of things becoming visible.

## Palette

| Variable | Value |
|----------|-------|
| Background | #0c0c12 |
| Surface | #13131e |
| Text | #e4e2dc — warm off-white |
| Accent | #AFA9EC — purple, used sparingly |
| Border | rgba(255,255,255,0.07) |

## Confidence colour temperature

Annotation underlines and pill tags use colour temperature to encode confidence:

| Confidence | Colour |
|-----------|--------|
| 85%+ | Warm orange #D4845A — high confidence burns bright |
| 70–84% | Amber #C9A06A — mid confidence |
| Below 70% | Muted warm grey #9A9078 — lower confidence recedes |

## Typography

- **Display** — Georgia, for key statements
- **Body** — System sans-serif, 17px, line-height 1.75
- **Mono** — Courier New or JetBrains Mono, for pattern names and technical output

## The annotation

An underline that draws itself beneath a phrase. A pattern tag that appears after it. On hover: a pattern card with name, category, definition, and confidence. No verdict. No recommendation. The structure returned. The conclusion yours.
